此次实践是利用 kali 自带的 ssh 服务器来对 ssh 进行一次初体验，对于 ssh 还有很多东西有待探索

简介

一种加密网络传输协议，可在不安全的网络中为网络服务提供安全的传输环境，用于远程登录，但是SSH服务仅限于终端程序。

目标

用本地主机远程登录kali机

过程

  1. kali安装ssh服务器

输入语句以安装ssh

sudo apt-get install openssh-server

查看安装是否成功

ssh -v

  1. 通过 ifconfig 查看kali机IP地址

  1. kali机开启ssh服务

service ssh start

通过下面命令查看ssh服务状态，状态应当是running

service ssh status

  1. 本机远程登录kali机

输入命令

ssh user@ip/hostname

首次进行登录时会出现如下信息

这是因为第一次登录时，本机对于服务器来说是陌生的，因此需要进行确认。

输入yes后再输入kali机密码就登录成功了

输入exit退出ssh

PS: 每次重新开启kali机后，ssh服务都是默认关闭状态，必须要重新开启该服务。
